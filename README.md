# 👩‍💼 HR Analysis Dashboard (Power BI)

## 🔹 Project Overview
This project analyzes HR data to understand employee trends, attrition, and workforce distribution.

## 🔹 Key Insights
- Employee Attrition Analysis
- Department-wise Performance
- Salary Distribution
- Gender Diversity Analysis
- Job Role Insights

## 🔹 Tools Used
- Power BI
- Excel / CSV
